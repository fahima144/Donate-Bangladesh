What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.
Web page is a document that can be either displayed in the browser window or as the HTML source. In both cases, it is the same document but the Document Object Model (DOM) representation allows it to be manipulated. As an object-oriented representation of the web page, it can be modified with a scripting language such as JavaScript.

How do you select an element from the DOM?

We can select an element from the DOM using JavaScript methods like getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), and querySelectorAll(). For example, document.getElementById('myId') selects the element with the ID "myId". The querySelector() method allows selecting the first matching element based on CSS selectors, such as document.querySelector('.myClass') for an element with the class "myClass". If we want to select multiple elements, querySelectorAll() returns a NodeList of all matching elements. Once selected, we can manipulate these elements, such as changing their content or styles.

What is event delegation in the context of the DOM, and why is it useful?

The process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.

How do you manipulate an element's attributes and styles using the DOM?

The classList object only provide methods to change the class of an element. If you want to change other attributes like id , href , or src , you can use the setAttribute() method. The setAttribute() method accepts two arguments: The name of the attribute to set.
